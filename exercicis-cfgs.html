Lloc web d'espectacles

L'exercici consistia a crear un lloc web fent servir HTML5, CSS i Bootstrap sobre Espectacles, amb una pàgina inicial, que estigués enllaçada a una pàgina de mostra d'algun dels articles, alguna notícia... El meu lloc web té una pàgina inicial, pàgines d'exemple d'espectacles (amb les seves pàgines de detall), dues o tres notícies (amb les pàgines de detall), una pàgina d'inici de sessió, de registre, de contacte i de compra d'entrades.
